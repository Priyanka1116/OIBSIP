# Data Cleaning

## Objective
To clean and prepare a raw dataset by handling missing values, removing duplicate records, correcting data types, and treating inconsistent data to make it suitable for analysis.

## Tools Used
- Python
- Pandas
- NumPy
- Google Colab

## Dataset
Raw dataset containing missing values, duplicate records, and inconsistent data.

## Steps Performed
- Loaded the dataset
- Checked dataset information
- Identified missing values
- Removed duplicate records
- Corrected data types
- Handled inconsistent values
- Renamed columns (if required)
- Saved the cleaned dataset

## Outcome
The dataset was successfully cleaned and transformed into a consistent, accurate, and analysis-ready format.

Completed as part of the Oasis Infobyte Data Analytics Internship.
