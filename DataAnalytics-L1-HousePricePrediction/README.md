# House Price Prediction

## Objective
To build a machine learning model that predicts house prices based on various features such as area, number of bedrooms, bathrooms, and other property details.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## Dataset
House Price Dataset containing property features and corresponding house prices.

## Steps Performed
- Loaded the dataset
- Performed data cleaning
- Checked for missing values and duplicates
- Conducted Exploratory Data Analysis (EDA)
- Selected important features
- Split the dataset into training and testing sets
- Trained a Linear Regression model
- Predicted house prices
- Evaluated the model using MAE, MSE, RMSE, and R² Score

## Outcome
Successfully developed a machine learning model to predict house prices and evaluated its performance using standard regression metrics.
